About App

This platform digitizes vehicle buying and selling processes, providing users with a fast, secure, and user-friendly experience. Developed with modern web technologies, our system meets the needs of both individual and corporate users.

Thanks to advanced filtering, role-based access control, and secure session management, each user only accesses the areas they are authorized to access. Our interface is responsive and intuitive, working seamlessly on desktop and mobile devices.

The application is built with Spring Boot and Angular infrastructure. It offers a sustainable software experience with JWT-based authentication, dynamic UI components, and a modular architecture.
